# Ship's Crane Operating Guide

This minimal web page provides a professional quick-reference for operating a ship's crane, covering hoisting, luffing, slewing, hydraulic discipline, and shutdown steps. It also includes a simple simulator with safety-focused logging.

## Usage
Open `index.html` in a modern browser. Use the buttons to run a pre-operation checklist, simulate hoist and transfer cycles, and practice command sequencing with an emergency stop.
